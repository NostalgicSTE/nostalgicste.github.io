                    Continuation
                

                

                    

                        

                        

                            
Active Directory

                            
Active Directory and dependant applications were one of the earliest software that we got
                                familiarized to upon
                                initiating the internship.Within the Active Directory software, I learnt how to setup
                                basic services such as SMTP, NTP and Static routing for
                                proper establishment of the server. This was a new experience for me, as this was all
                                new to me, from using Windows
                                Server, to the different services’ initiation. Through this portion of work, I got to
                                understand various protocols and services that run on our devices on a
                                split-second, and understood the work
                                that goes into the setup of them; These include, but are not limited to Simple Network
                                Management Protocol (SNMP),
                                Simple Mail Transfer Protocol (SMTP), Telnet, and Dynamic Host Configuration Protocol
                                (DHCP).

                            

                            
                        


                        

                            
Bitbucket

                            
Having used a similar product as Bitbucket, I instantaneous understood its workflow and
                                nuances of the platform. It was easy for me execute the migration of projects, which was
                                mostly automated by the software. The efforts come in during the pre-migration period,
                                where we had to look through every single one of the 80+
                                projects, and the repositories that they contain, and make
                                spreadsheet, containing the different access rights, description of projects, and
                                corresponding users. Additionally, we
                                had to cross check with Active Directory’s user list, to verify if the user was still
                                actively contributing to the
                                company or can be demoted from all project roles. All those above-mentioned activities
                                were to be planned to be done
                                across a span of 3 days, with some project prioritisation, as they were more urgent than
                                the other projects.
                            

                        


                        

                            
Network+

                            
I always loved to fantasize about networking systems, and the world of wireless
                                communications, since my secondary school days. Network+ by CompTIA is a great starting
                                point into this world, as it went through all from basics such as OSI (Open Systems
                                Interconnection) model and network topologies, to advanced concepts such as CIA Triad
                                and social engineering. I would say that this course was one of the key reasons I
                                applied for this internship.

                                    User Retrieval Executable
                                

                                

                                    User Retrieval executable file is a PowerShell executable cmdlet, created with the
                                    purpose of retrieving Active Directory
                                    user information and format it into an Excel-based file format. This project has
                                    gone through multiple alteration and
                                    modifications due to unforeseen circumstances, but was also one of the easiest tasks
                                    undertaken at work.
                                    


                                    This project initially was not the “User Retrieval Executable”, but was similar to
                                    the above mentioned “Jira Service
                                    Desk” application, referred to as the “User Information Requestable – Jira”. It was
                                    initially planned to be programmed
                                    in Jira, with a similar web interface as the service desk, but instead of multiple
                                    sub menus, it was supposed to serve
                                    one purpose, which is to accept an argument of username, and then return the user
                                    the information requested on the issue
                                    request page itself. However, it was not possible to execute an action of such
                                    complexity in Atlassian server. It was
                                    later changed to the now existing project “User Retrieval Executable”.
                                

                            

                            

                                
                                
                            

                            

                                

                                    Server installations
                                

                                

                                    we were involved in server configuration operations such as iLO setup, raiding of
                                    hard disks, switch management and not to forget wire works. in a later date, we also
                                    installed ubuntu OS into those servers. This portion of our internship was by far
                                    the most contentful and informative task. From learning about basics of how hard
                                    disks work, to understanding the importance of terminologies used in the networking
                                    field, things learnt during this task would be the most helpful in the future, in
